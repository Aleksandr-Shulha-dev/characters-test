# Character-app

## Installation
 - Run ```npm run install:all```

## Run app 

1. First run ```npm run start:backend```
2. Second run ```npm run start:frontend```
3. Open ```http://localhost:3000/```

## Database 
- lowdb

## Solutions BACKEN

- Typescript
- Node.js
- Express.js
- Multer
- Nodemon
